# Personal Portfolio - Astro Practice Project

Hi! This repository contains the source code for my personal web portfolio, developed by José Fabián.

This is a practice project created specifically to learn, master the basics, and explore the advantages of the **Astro** framework—going from a beginner level to building a fully functional and optimized product.

## Tech Stack

This project is built and styled using the following technologies:

- **Astro:** Main framework focused on speed and static content delivery.
- **Tailwind CSS:** For agile, responsive design and UI development.
- **HTML & CSS:** For the base structure and global configurations.

---

## Project Structure

Inside this Astro project, the main folder structure looks like this:

```text
/
├── public/
│   └── favicon.svg
├── src/
│   ├── components/       # Reusable components (Buttons, Icons, Header, etc.)
│   ├── layouts/          # Base templates (The layout wrappers)
│   ├── styles/           # Global CSS files (Tailwind configuration)
│   └── pages/            # Web pages (index.astro, projects.astro, etc.)
└── package.json
```
